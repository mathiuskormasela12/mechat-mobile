# MeChat
MeChat is social media application that is written in React Native.

## How To Run This App ?
- Make sure you had clone this repo
- Open your terminal in this project and type npx yarn install
- then run npx yarn start
- then run npx yarn android to start app to android device
- then run npx yarn ios to run app to iOS simulator
